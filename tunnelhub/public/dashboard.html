TunnelHub Guide

TunnelHub come's with a guide. Here you can see the manuals on how to use TunnelHub effectively. This guide only show the most basic and efficient way to use the service. For more advanced guide, please ping our devs or visit PHCorner.net

How it Works?

In a short word, TunnelHub will create a tunnel on your device. In long word, TunnelHub is just like OpenVPN, but it injects your OpenVPN file with a HTTP Payload.

Settings Up

If you're first time of using this app, Set up your VPN configuration first at the Settings page.

The first thing to do is to navigate to OpenVPN tab and upload your .ovpn config. After you've uploaded the config, put your username and password on the form and hit SAVE. Please note that OpenVPN UDP Protocol does not support http-proxy therefore, it does not support injections.

The next thing to do is to setup your injection at the HTTP Tab

HTTP Injection

HTTP Header are injected into your openVPN to connect into tunnel. There are two types of injection we can use.

HTTP
HTTP Payload are headers instruction that is injected to your OpenVPN. You can start injecting HTTP by choosing the Payload in the tunnel type dropdown.

The next thing to do is to generate your payload. Unfortunately, we can't generate a payload in this UI so you need to generate your payload using our recommended generator - HTTP Injector (for Mobile) or SVL Injector for PC.

HTTP Payload may look like this
CONNECT [host_port] [protocol][crlf]Host: gstatic.com[crlf]X-Online-Host: gstatic.com[crlf]X-Forward-Host: gstatic.com[crlf]Proxy-Connection: Keep-Alive[crlf]User-Agent: [ua][crlf][crlf][crlf]

SNI
SNI are more simpler injection method but yet powerful. To use SNI as an injection, make sure your .ovpn config supports SSL/TLS. To input your SNI Payload, select first the tunnel type dropdown at the Settings HTTP Tab and choose SNI, then put your SNI Payload into the textarea.

After all of the configurations that you have set, you are now ready to click the SAVE button

Starting up the VPN

After you've set all the things like a pilot, you are now ready to take-off and start your OpenVPN. It's just so easy to use. You just need to go back to the Dashboard and click the CONNECT button. Wait until it says connected and that's it. You can now surf the internet using the TunnelHub Connection

If you want to check the background logs while it's connecting, you can navigate to the Logs page by simply clicking at the top right icon. There you can see two logs, the OpenVPN log and the HTTP Log
